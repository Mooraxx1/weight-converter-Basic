# Weight Converter (Pounds to Kilograms)

A responsive, real-time weight conversion utility featuring a modern UI and 
automated error handling.



## 🌐 Live Demo
Visit the project here: 
https://mooraxx1.github.io/weight-converter-Basic/

## 🚀 Features
- **Instant Feedback:** Calculations occur as the user types.
- **Dynamic UI:** Built with a glassmorphism aesthetic and CSS gradients.
- **Error Management:** Validates inputs and clears errors via `setTimeout`.
- **Responsive:** Optimized for both desktop and mobile viewports.

## 🛠️ Technical Implementation
- **Logic:** JavaScript `input` event listener for low-latency updates.
- **Precision:** Results are rounded to 2 decimal places using `.toFixed(2)`.
- **Formula:** $$kg = \frac{lb}{2.2046}$$

## 📁 File Organization
- `index.html`: Contains the semantic structure and input fields.
- `style.css`: Manages the Flexbox layout and darkcyan/yellow color scheme.
- `index.js`: Handles the mathematical conversion and DOM manipulation.
